# ONeul(오늘)
One Day Lifetime SNS

## Development Goals
- Delete post 24 hours old using Spring Batch
- Managing Logins Using Session
- Object Oriented Programming 
- CQRS(Command and Query Responsibility Segregation) Structure
- RESTful API
